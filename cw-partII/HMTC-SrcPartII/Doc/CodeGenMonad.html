Description

Code generation monad. This module provides an abstraction for code generation computations with support for generation of distinct names, e.g. for labels.

Code generation computation

data CG i x a #

Code generation computation. Parameterised on the type of instructions and additional auxiliary information. One use of the auxiliary information is for additional separate output sections by instantiating with suitable disjoint union type.
